--------------------------------------------------------***Welcome to your Music Wrapped***--------------------------------------------------------

Music Wrapped is a custom built application that a user can login through Spotify and see a custom "Spotify Wrapped" during any time of the year.

***This application was not produced by, endorsed by, or partnered with Spotify***

I am an individual developer that built this application to showcase on my portfolio.

User authentication is handled with Spotify’s OAuth 2.0 Authorization Code Flow with PKCE. 
This approach is well-suited for single-page applications because it provides a secure login flow without requiring a client secret to be stored in the browser.

This application follows Spotify's developer requirements via Spotify Developer Dashboard - https://developer.spotify.com/
Due to restrictions for individual developers there are a few things to note
  - Individual developers are only allowed to have 5 active users per application. Therefore on my Portfolio page there is a video demo of me using my personal spotify account.
    (If you would like to have me personally demo the application, or you yourself want access, please email me at Thenn0110@gmail.com and include your full name and
    email associated with your Spotify account)
  - API usage is restricted for individual developers
    - I am only able to retrieve the top 50 rows in any category
    - There are only 3 time ranges that can be pulled (Past month, past 6 months, and all time)
    - Audio samples can no longer be retrieved
    - Only certain metadata can be retrieved (No minutes listened, times listened to a song, etc.)
